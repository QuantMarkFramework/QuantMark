# QuantMark

## Installation (Linux, requires python3 and pip)
```$ python3 -m venv venv```

```$ source venv/bin/activate```

```$ (venv) python3 -m install flask``` 

```$ export FLASK_APP=app.py```

```$ flask run```

The application now runs in localhost:5000

to exit venv: ```$ deactivate```
